# Catch Apple üçé

A simple 2D game made using Python and Pygame. Just a small game to pass your time and relax !
Catch apples from four different trees with the help of given instructions.

## Tech stack
- Python
- Pygame

## How to Run

### Prerequisites
1. python version 3.8 or above
2. pip

### Steps

1. Clone the repository
2. Install dependencies:
```bash
pip install pygame
```

### 3. run the game
```bash 
python CatchApple.py
```

## Controls
*A/D* - Move
*SPACE* - Start
*ESC* - Pause / Continue / Restart / Quit
